## ğŸ§  Overview

**Enzyme Activity Visualizer** is a Python project for **Class 11 Biology students**.  
It simulates how **enzymes respond to changes in temperature and pH** using Gaussian models and visualizes the results using Matplotlib.

Key functionalities include:

* Enzyme activity vs **temperature**
* Enzyme activity vs **pH**
* Adjustable **optimum conditions** and sensitivity
* Visualize activity curves for better understanding
